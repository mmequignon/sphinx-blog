pg-cliÂ¶

Just a wrapper of the pgcli postgres client.

Usage:

pg-cli <container-name> <db-name>

Requirements :

  • pgcli
  • a /tmp/pg_table file filled by the lxc-up script.
  • SSH properly configured.

It allows me to connect on a remote database, available on localhost through a SSH tunnel.

It uses the /tmp/pg_table created by the lxc-up script to get the port to connect on. It gets the username to connect with in your ~/.ssh/config file.
#!/bin/bash

PGCLI="/usr/bin/pgcli"
PG_TABLE="/tmp/pg_table"
AWK="/usr/bin/awk"

function get-user {
    echo "$($AWK '/'$1'/ {for(i=1; i<=4; i++) {getline ; print;}}' $HOME/.ssh/config | grep User | $AWK '{print $2}')"
}

function get-pg-port {
    echo $(grep $1 $PG_TABLE | awk '{print $2}')
}

pg_port=$(get-pg-port $1)
username=$(get-user $1)

$PGCLI -U $username -h localhost -p $pg_port -d $2
